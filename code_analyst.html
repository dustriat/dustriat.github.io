A series of hands-on portfolio projects completed through Codecademy's Data Scientist: Analytics career path. Each project targets a specific skill set — from writing Python scripts and querying databases to applying statistical analysis and exploring machine learning fundamentals.

About This Path

The Codecademy Data Scientist: Analytics career path is a structured, project-based curriculum covering the full scope of modern data science. Projects progress from foundational Python and data manipulation to advanced statistical methods and introductory machine learning.

Rather than a single capstone, this path produces a collection of portfolio-ready projects — each demonstrating a distinct, employer-relevant skill. All projects are published to GitHub as they are completed.

Project Modules

01

Python Fundamentals & Data Manipulation

Core Python scripting, working with Pandas and NumPy to load, inspect, and transform datasets.

02

SQL & PostgreSQL Queries

Database querying with PostgreSQL — joins, aggregations, subqueries, and window functions applied to structured datasets.

03

Data Visualization

Building charts and dashboards using Matplotlib, Seaborn, and other visualization libraries to communicate findings effectively.

04

Statistical Analysis

Hypothesis testing, probability distributions, regression analysis, and interpreting statistical significance.

05

Machine Learning Fundamentals

Introduction to supervised and unsupervised learning — classification, regression, and clustering using scikit-learn.
