Personal Portfolio
Website

Built entirely from scratch using HTML, CSS, and vanilla JavaScript — no frameworks, no templates. Designed to reflect professional identity, communicate career growth, and serve as a living document updated alongside skills and certifications. Hosted for free on GitHub Pages.

Project Overview

This portfolio site is itself a project — built to demonstrate initiative, technical capability, and design sense. Rather than using a template or site builder, every section was coded by hand using semantic HTML5, custom CSS with a design system of variables and utilities, and vanilla JavaScript for interactions and animations.

The site is designed to grow alongside a career in transition: sections for active projects, in-progress certifications, a full career timeline, and a skills system with proficiency indicators. Everything is easy to update as new work and credentials are added.

What Was Built

01

Design System & Color Palette

A split-complementary palette (#66B8FF, #666BFF, #66FFFA, #FF666B, #FFFA66) implemented as CSS custom properties with consistent typography using Barlow, DM Sans, and DM Mono.

02

Multi-Section Single Page

Hero with headshot and contact strip, contact form, projects, certifications, career timeline, and skills — all on one scrollable page with smooth anchor navigation.

03

Project Subpages

Four linked project detail pages (including this one) sharing the same stylesheet and script for visual consistency across the site.

04

JavaScript Interactions

Scroll-triggered reveal animations, sticky nav with blur effect, active link highlighting, mobile hamburger menu, and a mailto-powered contact form.

05

GitHub Pages Deployment

All files committed to a public GitHub repository and deployed via GitHub Pages — zero hosting cost, version-controlled, and publicly accessible.
